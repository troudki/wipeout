The Value attribute

wo-value="value"

data-wo-value="value"

The value attribute defines the html value of an element. The value attribute works best with the &lt;input>, &lt;textarea>, &lt;select> and &lt;option> tags.

For checkboxes and radio buttons see:

&lt;input>

<wo.content model--s="Hello Wipeout!"> <set-template> <input wo-value="$model" /> </set-template> </wo.content> <wo.view template-id="'wo-value-attr-input'"></wo.view>

&lt;textarea>

<wo.content model--s="Hello Wipeout!"> <set-template> <textarea wo-value="$model"></textarea> </set-template> </wo.content> <wo.view template-id="'wo-value-attr-textarea'"></wo.view>

&lt;select>, &lt;option>

<wo.content model="1"> <set-template> Model: <span wo-content="$model"></span> <br /> <select wo-value="$model"> <option wo-value="1">Option 1</option> <option wo-value="2">Option 2</option> <option wo-value="3">Option 3</option> <option wo-value="4">Option 4</option> </select> </set-template> </wo.content> <wo.view template-id="'wo-value-attr-select'"></wo.view>
