The Visible attribute

wo-visible="value"

data-wo-visible="value"

The visible attribute defines whether an element is visible or not.

<wo.content model="true"> <set-template> <span wo-visible="$model">This is visible</span>
<span wo-visible="!$model">This is not visible</span> </set-template> </wo.content> <wo.view template-id="'wo-visible-attr'"></wo.view>
