Observe the value of the property

If you need to observe the property value for change, you can use the onPropertyChanged function.

Declaration: <wo.view model--mybinding="$model"></wo.view> Binding: wo.bindings.mybinding = function mybinding (viewModel, property, renderContext) { // will observe the model property of the view model for change property.onPropertyChanged(function (oldValue, newValue) { // Do any binding stuff here }, true); };

The second argument to onPropertyChanged tells wipeout to execute the callback immediately, as if the property had just changed.
