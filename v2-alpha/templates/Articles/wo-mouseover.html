The Mouseover attribute

wo-mouseover="eventHandler"

data-wo-mouseover="eventHandler"

The mouseover attribute defines an event handler to be fired when a mouse is moved within the element.

<wo.content model--s="Hello Wipeout!"> <set-template> <input wo-mouseover="logic: alert($model)" /> </set-template> </wo.content> <wo.view template-id="'wo-mouseover-attr'"></wo.view>
