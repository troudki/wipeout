The Change attribute

wo-change="eventHandler"

data-wo-change="eventHandler"

The change attribute defines an event handler to be fired when the element value changes.

<wo.content model--s="Hello Wipeout!"> <set-template> <input wo-change="logic: alert($model)" /> </set-template> </wo.content> <wo.view template-id="'wo-change-attr'"></wo.view>
