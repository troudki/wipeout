Rendering Simple Content

There are a few ways to display content contained in a view model

1. Using {{...}}

You can render anything by putting it within {{...}} parenthesis. For example to render the model of the current templated view model.

<div> <h1>The model</h1> {{$this.model}} </div>

You can change the {{...}} parenthesis to anything else using the renderParenthesis function. Remember that this must be done before any view models are rendered in you application in order to remove inconsistencies.

wipeout.template.rendering.compiledTemplate.renderParenthesis("[[", "]]");

2. Using wo-render

The html attribute does almost the same as using {{...}}, the difference being that the content will be surrounded by a html tag.

<div> <h1>The model</h1> <span wo-render="$this.model"></span> </div>

3. Using wo-content

The html attribute does almost the same as using wo-render, the difference being that:

  1. wo-content is meant for rendering simple values such as strings and numbers. It cannot be used to render full view models.
  2. wo-content is lightweight.

<div> <h1>The model</h1> <span wo-content="$this.model"></span> </div>
