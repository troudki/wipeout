Get the value of the setter

To get the setter value use the getter function to create a getter and use the getter to get the value.

Declaration: <div wo-my-attr="$model"></div> Attribute: wo.addHtmlAttribute("my-attr", function myattr (element, attribute, renderContext) { // returns a function which can be used to get the value var getter = attribute.getter(); // returns the value var theValue = getter(); });

If you are using some asynchronus code (i.e. defining any functions within the attribute), it is important to cache the getter. The attribute.getter() function will only work synchronusly, however the result of the getter function can be invoked at any time

wo.addHtmlAttribute("my-attr", function myattr (element, attribute, renderContext) { // cache the getter here var getter = attribute.getter(); // create an asynchronus function which will be called when the value changes attribute.watch(function (oldValue, newValue) { // use the getter here var result = getter(); }); };
