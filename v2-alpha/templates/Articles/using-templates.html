Templates

Templates are referenced by the templateId property of the view model. The template can be set or re-set at any time over the view model lifecycle. For instance:

Given the view model: var viewOrEdit = wo.viewModel("demoApp.viewOrEdit") .value("text", "Edit") .value("model", "Hi!") .templateId("view-or-edit-view") .build(); viewOrEdit.toggle = function () { if (this.templateId === "view-or-edit-view") { this.templateId = "view-or-edit-edit"; this.text = "View"; } else { this.templateId = "view-or-edit-view" this.text = "Edit"; } }; And templates:
view-or-edit-view <span wo-content="$model"></span><br /> <button wo-click="$this.toggle" wo-content="$this.text"></button> view-or-edit-edit <input type="text" wo-value="$model" /><br /> <button wo-click="$this.toggle" wo-content="$this.text"></button> Result: <demo-app.view-or-edit></demo-app.view-or-edit>
