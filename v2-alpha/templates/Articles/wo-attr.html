The Attribute attribute

wo-attr-attrName="value"

data-wo-attr-attrName="value"

The attr attribute defines an attribute on a html element. The attribute name will be the end of the wo-attr- definition.

<wo.content model--s="Hello Wipeout!"> <set-template> <input wo-attr-value="$model" /> </set-template> </wo.content> <wo.view template-id="'wo-attr-attr'"></wo.view>
