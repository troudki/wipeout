Hello World

Wipeout goes with the idea, "If I can create it, I can render it", so to create a basic wipeout app, you simply need to create a class, or in the case of javascript, a function

function helloToTheWorld() { // for simple objects, wipeout likes to render the result of the "toString" method. this.toString = function () { return "Hello wipeout!"; }; }

With helloToTheWorld in place it is just a case of adding a <hello-to-the-world> definition to your html.

Creating the application

Within the html document, we now define our viewModel.

<hello-to-the-world> </hello-to-the-world>

What wipeout will do here is:

  1. Scan the DOM for nodes which it can find on the window object. In this case "hello-to-the-world" will be camel cased to "helloToTheWorld" and because window.helloToTheWorld exists, this node is marked as a viewModel.
  2. Create an instance of helloToTheWorld, i.e. var theViewModel = new helloToTheWorld();
  3. Use the toString method to find the rendered content, i.e. var html = theViewModel.toString();

This will ultimately render: Hello wipeout!

The full code for this example is as follows

<!DOCTYPE> <html> <head> <title>Hello Wipeout</title> </head> <body> <hello-to-the-world></hello-to-the-world> <!-- Required script --> <script type="text/javascript" src="wipeout.js"></script> <!-- The application viewModel --> <script type="text/javascript"> function helloToTheWorld() { // for simple objects, wipeout likes to render the result of the "toString" method. this.toString = function () { return "Hello wipeout!"; }; } </script> </body> </html>
