Event Attributes

Event attributes are attributes which fire off a html event. One example of this is .

With event attributes you can define the event handler in multiple ways:

As a logical string

If you prefix the click attribute value with logic:, the contents of the attribute will be executed.

<button wo-click="logic: console.log('The button was clicked')">Click me</button>

As an event handler

You can point the event at an event handler anywhere within the current render context.

<button wo-click="$parent.logButtonClick">Click me</button>

As an event handler with arguments

You can point the event at an event handler anywhere within the current render context and include arguments.

<!-- e is the html event arguments, element is the html element --> <button wo-click="$this.logButtonClick($model, e, element, etc...)">Click me</button>
