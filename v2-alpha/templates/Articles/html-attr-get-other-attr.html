Access another wipeout attribute

Some wipeout attributes depend on one another, and you may wish to access one attribute within another. You can do this with the function.

Declaration: <input type="text" wo-my-attr="$model" wo-value="$model.property1" /> Attribute: wo.addHtmlAttribute("my-attr", function myattr (element, attribute, renderContext) { var getter; var success = attribute.otherAttribute("value", function (value) { // value is the "wo-value" attribute for this element getter = value.getter(); }); if (success) { var val = getter(); } });

The attribute.otherAttribute attempts to find the other attribute and returns true/false depending on it's success. If the attribute is found, the callback will be run and you can access the other attribute.
