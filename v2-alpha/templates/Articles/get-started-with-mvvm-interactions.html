MVVM Interactions

Expanding on the , there are several ways in which the layers (model, viewModel, template) interact with one another.

View model -to- template

Within wipeout, the view model can interact with the template via it's templateItems property.

View model: { visible: true, templateItems: {}, toggleVisibility: function () { this.visible = !this.visible; }, getNewName: function () { return this.templateItems.newName.value; }, model: { people: ["Mary", "John"], addPerson: function (name) { this.people.push(name); } }, templateId: "togglePeopleTemplate" } Template: <div> <button wo-click="$this.toggleVisibility">Toggle</button> <div wo-visible="$this.visible"> {{$model.people}} </div> <input type="text" id="newName" /> </div>

View model -to- model

The view model can alter the model directly

View model { visible: true, templateItems: {}, toggleVisibility: function () { this.visible = !this.visible; }, getNewName: function () { return this.templateItems.newName.value; }, addNewName: function () { var newPerson = this.getNewName(); this.model.addPerson(newPerson); }, model: { people: ["Mary", "John"], addPerson: function (name) { this.people.push(name); } }, templateId: "togglePeopleTemplate" } Template: <div> <button wo-click="$this.toggleVisibility">Toggle</button> <div wo-visible="$this.visible"> {{$model.people}} </div> <input type="text" id="newName" /> <button wo-click="$this.addNewName">Add</button> </div>
