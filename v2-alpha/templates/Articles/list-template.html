Setting the list template

List items are rendered unbounded by default. You can change this using the property of the itemSource, and rendering the $this.items property.

<wo.list items="['Hello wipeout 1', 'Hello wipeout 2']"> <set-template> <ul> {{$this.items}} </ul> </set-template> <item-template> <li> <span wo-content="$model"></span>, Index: <span wo-content="$index.value"></span> </li> </item-template> </wo.list> <wo.view template-id="'list-template-script-1'"></wo.view>
