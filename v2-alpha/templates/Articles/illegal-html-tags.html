Illegal html tags

In some areas, html has a strict schema which disallows you placing certain tags within other tags. One such example is that a tbody tag can only be placed within a table tag. To get around this you can use the wo-el or data-wo-el attributes.

<!-- this will not work --> <wo.content> <set-template> <tbody> ... </tbody> </set-template> </wo.content> <!-- this will work --> <wo.content> <table wo-el="setTemplate"> <tbody> ... </tbody> </table> </wo.content>

The wo-el attribute changes the meaning of the element to wipeout, while still following html rules
