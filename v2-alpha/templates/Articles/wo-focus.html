The Focus attribute

wo-focus="eventHandler"

data-wo-focus="eventHandler"

The focus attribute defines an event handler to be fired when the element receives focus.

<wo.content model--s="Hello Wipeout!"> <set-template> <input wo-focus="logic: alert($model)" /> </set-template> </wo.content> <wo.view template-id="'wo-focus-attr'"></wo.view>
