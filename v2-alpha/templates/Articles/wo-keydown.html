The Keydown attribute

wo-keydown="eventHandler"

data-wo-keydown="eventHandler"

The keydown attribute defines an event handler to be fired when a key is pressed.

<wo.content model--s="Hello Wipeout!"> <set-template> <input wo-keydown="logic: alert($model)" /> </set-template> </wo.content> <wo.view template-id="'wo-keydown-attr'"></wo.view>
