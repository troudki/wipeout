The Mouseup attribute

wo-on-event="event name"

data-wo-on-event="event name"

When combined with another html attribute, determines when that attribute will be fired.

<wo.content model--s="Hello Wipeout!"> <set-template> <span wo-content="$model"></span> <br /> <input wo-on-event="blur" wo-value="$model" type="text" /> Model will change when you leave the textbox <br /> <input wo-on-event="keyup" wo-value="$model" type="text" /> Model will change when you release a key </set-template> </wo.content> <wo.view template-id="'wo-mouseup-attr'"></wo.view>
