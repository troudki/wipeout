The Checked Value attribute

wo-checked-value="value"

data-wo-checked-value="value"

The value attribute defines what value to set if a checkbox or radio button is checked. It can be used in conjunction with the or value attributes. If no value is provided a default value of true is used.

For checkboxes and radio buttons see:

Radio button

<wo.content model="1"> <set-template> Model: <span wo-content="$model"></span> <br /> <label>Value 1:</label> <input type="radio" wo-checked-value="$model" wo-value="1" name="radio" /> <br /> <label>Value 2:</label> <input type="radio" wo-checked-value="$model" value="2" name="radio" /> </set-template> </wo.content> <wo.view template-id="'wo-checked-value-attr-radio'"></wo.view>

Checkbox with value

<wo.content model--s="Hello Wipeout!"> <set-template> Model: <span wo-content="$model == null ? 'null' : $model"></span> <br /> <label>Value:</label> <input type="checkbox" wo-checked-value="$model" value="Hello Wipeout!" /> </set-template> </wo.content> <wo.view template-id="'wo-checked-value-attr-checkbox-1'"></wo.view>

Checkbox without value

<wo.content model="true"> <set-template> Model: <span wo-content="$model"></span> <br /> <label>Value:</label> <input type="checkbox" wo-checked-value="$model" /> </set-template> </wo.content> <wo.view template-id="'wo-checked-value-attr-checkbox-2'"></wo.view>
