Where do templates go?

Templates can go in one of three places

1. Embedded within the DOM

The first place wipeout will look for a template is in the DOM. It will do a document.getElementById(...) lookup where the template id is the id of the element. To use this method, you should place your template within script tags like so:

<script type="text/html" id="theTemplateId"> ... template code </script>

2. Loaded from the server

If wipeout cannot find a template in the DOM it will take the templateId as a URL and try to find it with an ajax call. So for instance using a templateId like http://www.mydomain.com/templates/myTemplate.html will also work.

You can disable this functionality by setting wipeout.settings.asynchronousTemplates = false;

3. Compile a template from a string

Once a wipeout template is found, it is compiled and cached. You can skip the entire lookup process by compiling templates yourself:

wipeout.template.engine.instance.setTemplate(templateId, templateString);
