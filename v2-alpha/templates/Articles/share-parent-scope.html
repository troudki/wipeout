Share parent scope

It is not always necessary to create a scope for each viewModel. A good example of this is in using the control, where the scope of the if istself is never needed. You might also notice yourself writing bindings like wo-value="$parents[4].value" which is an indicator of un-needed scope.

You can share the parents binding scope by setting share-parent-scope="true" in a template definition or as a value in the view model definition.

<wo.view share-parent-scope="true"></wo.view> - or - wo.viewModel("myViewModel") .value("shareParentScope", true) .build();

An example of scope

<wo.content the-scope="'Parent scope'"> <set-template> <div> <wo.content share-parent-scope="true" the-scope="'Child scope 1'"> <set-template> Scope is: <span wo-content="$this.theScope"></span> </set-template> </wo.content> </div> <div> <wo.content the-scope="'Child scope 2'"> <set-template> Scope is: <span wo-content="$this.theScope"></span> </set-template> </wo.content> </div> </set-template> </wo.content> <wo.view template-id="'share-parent-scope-script-1'" />

By default, the shareParentScope property of a control is true.
