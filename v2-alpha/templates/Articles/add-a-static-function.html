Add a static function

You can add a static function to any viewModel using the staticFunction function. Static functions are functions which are appended to the viewModel class, so for instance if you have a viewModel named myNameSpace.myViewModel and you add a staic function function myStaticFunction () { }, then myNameSpace.myViewModel.myStaticValue() will be valid.

wo.viewModel("myNamespace.myViewModel") .staticFunction("myStaticValue", function () { // do something }) .build();
