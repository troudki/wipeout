The list binding

The items control binding tells an viewModel where to render its items. It doesn't need an argument and will throw an exception if not used by an list.

<wo.list item-source="['Hello wipeout 1', 'Hello wipeout 2']"> <set-template> <ul data-bind="list: null"></ul> </set-template> <item-template> <li> <span data-bind="text: model"></span> Index: <span data-bind="text: $index"></span> </li> </item-template> </wo.list> <wo.view template-id="'list-binding-script-1'" />
