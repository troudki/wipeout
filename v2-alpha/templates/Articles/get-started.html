Getting started

Getting started is really easy. It's javascript easy! Just download the latest verion of wipeout and add it to your page.

Included in wipeout are two external libraries:

  • is a simple object oriented library for javascript. It is used under the hood in wipeout and can be used for advanced wipeout view model building scenarios.
  • is a client binding library which allows you to observe for changes to the property of an object. It is used extensively by wipeout and you will most likely use it's observe and computed functionality.
