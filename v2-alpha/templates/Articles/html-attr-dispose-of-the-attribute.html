Dispose of the attribute

There are often subscriptions and other disposable scenarios which occur outside of the attribute argument to the attribute. These must be disposed of manually to prevent memory leaks. In order to dispose of an item manually a dispose function must be returned from the attribute function.


Declaration <div wo-my-attr="$model"></div> Attribute: wo.addHtmlAttribute("my-attr", function myattr (element, attribute, renderContext) { // observe the model property of the viewModel. This will create a disposable // which we want to dispose of when the attribute is torn down var disposable = renderContext.$this.observe("model", function (...) { ... }); return function () { disposable.dispose(); }; -- OR -- return disposable; });
