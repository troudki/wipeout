The Event attribute

wo-event-eventName="value"

data-wo-event-eventName="value"

The event attribute defines a html event handler on an element. The event name will be the end of the wo-event- definition.

Many common events such as click and blur have their own html attributes. Either approach is equally valid.

<wo.content model--s="Hello Wipeout!"> <set-template> <button wo-event-click="logic: alert($model)">Click me</button> </set-template> </wo.content> <wo.view template-id="'wo-event-attr'"></wo.view>
