A list filter

The following example is of a list of people filtered by age

// add filters to wo.filters wo.filters.ageFilter = { // This filter only returns those properties whose // age is greater than the min age downward: function (rockstars, age) { if (!rockstars) return []; age = age || 0; var output = []; for (var i = 0; i < rockstars.length; i++) { if (rockstars[i].age >= age) output.push(rockstars[i]); } // sort by age output.sort(function (a, b) { return a.age > b.age; }); return output; } }; <wo.content min-age="40"> <model>[ {name: "John", age: 25}, {name: "Paul", age: 2}, {name: "George", age: 81}, {name: "Ringo", age: 44}, {name: "Mick", age: 65}, {name: "Keith", age: 13}, {name: "Ronnie", age: 52}, {name: "Charlie", age: 30} ]</model> <set-template> <div style="min-height: 200px;"> Filter by age: <input type="text" wo-value="$this.minAge" wo-on-event="keyup" /> <wo.list items="$model, $this.minAge => ageFilter"> <item-template> <div wo-content="$model.name + ', ' + $model.age"></div> </item-template> </wo.content> </div> </set-template> </wo.content>

In the above example, the filter is defined as follows:

  1. The left hand side are arguments. All arguments will be passed into the filter.
  2. The argument after the => is the filter name.
