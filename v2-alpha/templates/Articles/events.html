Wipeout events

Models can communicate with view modes through the use of events.

wo.viewModel("myViewModel") .initialize(function () { this.registerEvent("model", "eventName1", function (eventArgs, model) { // the model has fired an event }); this.registerEvent("model.modelProperty", "eventName2", function (eventArgs, modelProperty) { // the modelProperty has fired an event }); }) .build();

In the example above the model has subscribed to the eventName1 event of it's model and the eventName2 event of the modelProperty of it's model. registerEvent is described .

To trigger an event you use the function.

var model = { pokeViewModel: function (eventArgs) { wo.triggerEvent(this, "eventName1", eventArgs); }, modelProperty: { pokeViewModel: function (eventArgs) { wo.triggerEvent(this, "eventName2", eventArgs); } } } model.pokeViewModel({}); model.modelProperty.pokeViewModel({});
