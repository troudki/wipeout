The Submit attribute

wo-submit="eventHandler"

data-wo-submit="eventHandler"

The submit attribute defines an event handler to be fired when a form is submitted.

<wo.content model--s="Hello Wipeout!"> <set-template> <form wo-submit="logic: alert($model)"> <input type="submit" value="Click me" /> </form> </set-template> </wo.content> <wo.view template-id="'wo-submit-attr'"></wo.view>
