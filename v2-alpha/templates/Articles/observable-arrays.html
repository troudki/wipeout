Observing Array Changes

Unlike regular objects, wipeout cannot modify Arrays behind the scenes so that they are observable. Instead you must create any arrays you need as observable arrays.

// this is incorrect var model = { innerArray: [1, 2, 3] }; // this is correct var model = { innerArray: wo.array([1, 2, 3]) };

In the first case, wipeout can bind to the array, but it will not be notified if the array changes.
In the second case wipeout can subscribe to array changes and update the DOM/view models accordingly.
