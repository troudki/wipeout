The Blur attribute

wo-blur="eventHandler"

data-wo-blur="eventHandler"

The blur attribute defines an event handler to be fired when the element blurs.

<wo.content model--s="Hello Wipeout!"> <set-template> <input wo-blur="logic: alert($model)" /> </set-template> </wo.content> <wo.view template-id="'wo-blur-attr'"></wo.view>
