Add a static value

You can add a static value to any viewModel using the staticValue function. Static values are values which are appended to the viewModel class, so for instance if you have a viewModel named myNameSpace.myViewModel and you add a staic value myStaticValue = 4, then myNameSpace.myViewModel.myStaticValue === 4.

wo.viewModel("myNamespace.myViewModel") .staticValue("myStaticValue", 4) .build();

Unlike non static values, it does not matter if the value is a primitive or not.
