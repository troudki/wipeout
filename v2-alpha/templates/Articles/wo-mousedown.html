The Mousedown attribute

wo-mousedown="eventHandler"

data-wo-mousedown="eventHandler"

The mousedown attribute defines an event handler to be fired when a mouse button is pressed.

<wo.content model--s="Hello Wipeout!"> <set-template> <input wo-mousedown="logic: alert($model)" /> </set-template> </wo.content> <wo.view template-id="'wo-mousedown-attr'"></wo.view>
