Accessing html elements and view models

You can access any view models and html elements defined in a template by giving them an id and using the templateItems property.

Template: <div id="theDiv"></div> <wo.view id="theView"></wo.view> View model: wo.viewModel("myViewModel") .rendered(function () { var div = this.templateItems.theDiv; var view = this.templateItems.theView; }) .build();
