Wipeout namespaces

Wipeout has two root namespaces which you can use interchangeably. The root namespace is window.wipeout which contains all of the public and private functionality of the library.

There is a second namespace window.wo which is an amalgamation of the window.wipeout.base and window.wipeout.utils namespaces, and is meant as a convenient alias for its longer namespace counterparts.
