Working with lists

Within wipeout, lists are handled by the viewModel. The items control allows you to specify a template for each item in the list and render accordingly.

<wo.list items="['Hello wipeout 1', 'Hello wipeout 2']"> <item-template> <div> <span wo-content="$model"></span>, Index: <span wo-content="$index.value"></span> </div> </item-template> </wo.list> <wo.view template-id--s="working-with-lists-1"></wo.view>

In the itemTemplate, you access an object from the items using the $model property. You can also access the object's index using $index.value.
