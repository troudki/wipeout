The Keypress attribute

wo-keypress="eventHandler"

data-wo-keypress="eventHandler"

The keypress attribute defines an event handler to be fired when a key is pressed and released.

<wo.content model--s="Hello Wipeout!"> <set-template> <input wo-keypress="logic: alert($model)" /> </set-template> </wo.content> <wo.view template-id="'wo-keypress-attr'"></wo.view>
