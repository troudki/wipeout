Setting the global binding strategy

The global binding strategy is the default way bindings will be handled for all view models you can set the global binding strategy like so:

wipeout.settings.bindingStrategy = wipeout.settings.bindingStrategies.createObservables; // - or - wipeout.settings.bindingStrategy = wipeout.settings.bindingStrategies.bindNonObservables; // - or - wipeout.settings.bindingStrategy = wipeout.settings.bindingStrategies.onlyBindObservables;
