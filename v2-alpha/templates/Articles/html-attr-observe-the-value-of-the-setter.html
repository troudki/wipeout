Observe the value of the setter

If you need to observe the setter for change, you can use the watch function.

Declaration: <div wo-my-attr="$model"></div> Attribute: wo.addHtmlAttribute("my-attr", function myattr (element, attribute, renderContext) { attribute.watch(function (oldVal, newVal) { // code for when the model value of the parent changes here }, true); });

The second argument to watch tells wipeout to execute the callback immediately, as if the property had just changed.
