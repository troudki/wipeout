Pluggable MVVM components

Expanding on the , MVVM screams componentization. Wipeout has a few view model types for basic scenarios, and you can also create your own.

In this example we will use the component to render the person names in a &lt;ul>. You can set properties of a nested component within the template.

<div> <button wo-click="$this.toggleVisibility">Toggle</button> <ul wo-visible="$this.visible"> <wo.list items="$model.people"> <item-template> <li> {{$model}} </li> </item-template> </wo.list> </ul> <input type="text" id="newName" /> <button wo-click="$this.addNewName">Add</button> </div>
