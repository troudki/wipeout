Set a string property

By default properties without a type are parsed as strings. You can also explicitly set invoke the string parser.

<wo.content> <my-val constructor="string"> Hello wipeout! </my-val> <set-template> Binding to element value: <span data-bind="text: myVal"></span> </set-template> </wo.content> <wo.view template-id="'set-string-property-script-1'" />
