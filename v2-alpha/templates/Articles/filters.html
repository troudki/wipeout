Binding filters

Binding filters are an extremely powerful way of altering a property value before the parent binding updates are executed. Filters can be used for both viewModel and html element properties. Binding filters are used like this:

<wo.view filtered-value="$this.value => myFilter"></wo.view>

Filters are added to wo.filters and have 2 functions.

  1. The downward function describes how to alter a value when setting the child property.
  2. The upward function describes how to alter a value when setting the parent property.
Both functions are optional.

wo.filters.exampleFilter = { downward: function (input1, input2, etc...) { // use this filter to alter values from a view model // before they are applied to a html element or child view model return alteredValues; }, upward: function (input1, input2, etc...) { // use this filter to alter values from a html element or // child view model before they are applied to the parent. return alteredValues; } }
