Subscribe to an element event

To subscribe to a html event of the element, use the function

Declaration: <div wo-my-attr="$this.doSomething()"></div> Attribute: wo.addHtmlAttribute("my-attr", function myattr (element, attribute, renderContext) { // method 1, using the event handler in the attribute. // when the div is clicked $this.doSomething() will be invoked attribute.onElementEvent("click"); // method 2, using another event handler. // when the div is clicked the callback will be executed attribute.onElementEvent("click", function () { // event handler logic }); });
