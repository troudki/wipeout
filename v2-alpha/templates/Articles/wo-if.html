The If attribute

wo-if="value"

data-wo-if="value"

The if attribute renders the content of an element only if the content of the attribue evaluates to true.

The if attribute wraps an , which can be used if you also need to define an else template.

<wo.content model="true"> <set-template> <span wo-if="$model">This will be rendered</span> <span wo-if="!$model">This will not be rendered</span> </set-template> </wo.content> <wo.view template-id="'wo-if-attr'"></wo.view>
