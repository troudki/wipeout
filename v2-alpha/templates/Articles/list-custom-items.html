Using custom items control items

With a custom list you can control the types of items which are created by overriding the function.

var mylist = wo.viewModel("mylist", wo.list) .build(); mylist.createItem = function(model) { var item = new myViewModelType(); item.model(model); item.templateId(this.itemTemplateId()); return item; }; Usage: <my-list items="...">...</my-list>
