Binding Setters

There are many values available to you when setting view model properties or using .

  • $this references the view model which is being templated
  • $model references the model of the view model being templated
  • $parent references view model 1 level up
  • $parents is an array of all view model ancestors where $parents[0] === $parent.
  • $parentContext is the binding context of the $parent.
  • $index.value is available to items in an and represents their index in the array.
  • is a tool to find ancestors based on search terms.

You can put almost any code you want into a binding setter

Examples: <!-- bind to the model --> <wo.view model="$model"></wo.view> <!-- bind to the value property of the view model --> <wo.view model="$this.value"></wo.view> <!-- bind to the value property of the parent view model --> <wo.view model="$parent.value"></wo.view> <!-- bind to a dynamic value --> <wo.view model="$model != null ? true : false"></wo.view> <!-- bind to a global value --> <wo.view model="document"></wo.view> <!-- find a view model and bind to it's model --> <wo.view model="$context.find(myViewModel).model"></wo.view>
