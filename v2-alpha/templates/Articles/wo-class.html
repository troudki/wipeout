The Class attribute

wo-class-className="value"

data-wo-class-className="value"

The class attribute defines an switches a html class on or off for a given element. The class name will be the end of the wo-class- definition.

<wo.content model="true"> <set-template> <span wo-class-green="$model">This class is applied</span>
<span wo-class-green="!$model">This class is not applied</span> </set-template> </wo.content> <wo.view template-id="'wo-class-attr'"></wo.view>
