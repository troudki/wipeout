Set the value of the setter

You may wish to set the value within the attribute setter. If this is the case, ensure you check this is possible with the canSet method and then use the setter method to set the value.

Declaration: <div wo-my-attr="$model"></div> Attribute: wo.addHtmlAttribute("my-attr", function myattr (element, attribute, renderContext) { // It IS possible to set the property "$model". // It would NOT be possible to set something like "$model ? $model : null" if (attribute.canSet()) { // get a function which can be used to set the value var setter = attribute.setter(); // set the model of the parent to "Hello!" setter("Hello!"); } });

If you are using some asynchronus code (i.e. defining any functions within the attribute), it is important to cache the setter. The attribute.setter() function will only work synchronusly, however the result of the setter function can be invoked at any time. See the example for a for details.
