Manually create a wipeout application

If you include the wipeout script in your page, wipeout will automatically scan the DOM and attempt to create view models wherever it can. Sometimes however, the viewModels are not available at the beginning of the page lifecycle, and you will need to set off the process yourself. You can do this with the wo() function.

Arguments

Name Description Default
model The root model for the application null
element The root element to begin searching for view models. You can also enter a string, in which case it will be the id of the root element body

Examples

// find and create all view models wo(); // find the element #myViewModel and convert it to a view model wo(null, "myViewModel"); // find the element #myViewModel and convert it to a view model // with a given model wo(model, "myViewModel"); // convert all elements to view models within the body wo(model, document.body);
