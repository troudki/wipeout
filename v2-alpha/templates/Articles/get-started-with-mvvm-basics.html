MVVM Basics

MVVM stands for Model, View, ViewModel. The meaning of the acronym has changed over the years, but the basic development concepts remain. You can read about MVVM here.

Within wipeout, MVVM aplications have three seperate and distinct types of object

1. View Models

View models are pieces of javascript code which

  1. Represent the state of something which has been rendered.
  2. Can modify the state of something which has been rendered.
  3. Don't really do much else. For instance ajax calls from a view model are generally to be avoided, as, whatever is happening on the server/external source is outside the domain of the visual.
An example view model might look like this:

{ visible: true, toggleVisibility: function () { this.visible = !this.visible; } }

2. Models

Models are pieces of javascript code which represent non visual data, or business data. Models have no concept of appearence. They have never heard of the html DOM or jQuery and should never use them. An example model might look like this:

{ people: ["Mary", "John"], addPerson: function (name) { this.people.push(name); } }

Models are connected to view models in that each view model has a model property. So now our view model looks like this:

{ visible: true, toggleVisibility: function () { this.visible = !this.visible; }, model: { people: ["Mary", "John"], addPerson: function (name) { this.people.push(name); } } }

3. Templates

Templates are pieces of html which describe the appearance of a view model. The template has access to the view model, and hence the model, as the model is a property on the view model. An example template might look like this:

<div> <!-- invoke the toggleVisibility function --> <button wo-click="$this.toggleVisibility">Toggle</button> <!-- control the visiblility --> <div wo-visible="$this.visible"> <!-- render the person names --> {{$model.people}} </div> </div>

Templates are connected to view models in that each view model has a templateId property. So now our view model looks like this:

{ visible: true, toggleVisibility: function () { this.visible = !this.visible; }, model: { people: ["Mary", "John"], addPerson: function (name) { this.people.push(name); } }, templateId: "togglePeopleTemplate" }
