Binding to global values

The binding scope also includes the window object, so you can bind to global variables

window.exampleValue = { innerValue: "Hello wipeout!" }; <wo.content model="exampleValue.innerValue"> <set-template> Binding to global: <span wo-content="$model"></span> </set-template> </wo.content> <wo.view template-id="'bind-to-global-script-1'"></wo.view>
