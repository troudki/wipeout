Add Computed Values

Computeds are a part of busybody, inspired by knockout computeds. There is full documentation on how they work here.

Computeds are baked into wipeout view models, and can be used like so:

wo.viewModel("myViewModel") .initialize(function () { // define computeds in the initialize function this.computed("fullName", function () { if (this.model) return this.model.firstName + " " + this.model.lastName; else return ""; }); }) .build();

Now you can bind to the fullName property in the template of the view model
